Productify is a full-stack product management platform designed for seamless inventory tracking and user-specific data management. Built with a focus on type safety and high performance, it bridges the gap between complex database relations and a clean, intuitive user interface.

🛠️ Tech Stack
Frontend: Next.js 14/15 (App Router), Tailwind CSS and DaisyUI

Backend: Node.js, TypeScript

Database: PostgreSQL with Drizzle ORM
